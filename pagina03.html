O CSS documento de estilização em cascata criado em 1993, com sua primeira versão amplamente utilizada em 1997. A linguagem CSS permite definir regras que definem caracteristicas para os elementos HTML como cor, borda e comportamentos, além de animações e transições.

Ao longo dos anos o css envoluio e hj temos o CSS 1 2 3, cada uma das versões adicionou novos recursos que eram demandados no periodo.

  1. CSS 01 1996 - propriedades básicas de estilização e formatação, além de controle do posicionamento dos elementos HTML copm display.
  2. CSS 02 1998 - adição de propriedades de animações e transição e melhoria do position.
  3. CSS 03 2008 - incremento gradual de novas funcionalidades do CSS com animações e ferramentas de posicionamento como display: flex (2008) e grid (2013)
